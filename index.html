Definition

Javascript is programming language initially created to cater for interactive webpages including online games, it is primary used in conjunction with HTML and CSS for frontend development.

All modern web browsers do support Javascript without requiring plug-ins. Javascript purpose is to follow instructions defined by the programmer in a pre-defined senario.

JavaScript is a:
- High-level (machine language), are languages that comprise variables, arrays, objects, complex arithmetic, boolean expressions, functions, loops, threads and locks.
- Dynamic, types are associtated to values e.g. a variable definition can evolve. It can first be defined as a string then as a number.
- Weakly typed, are more lient then strongly typed languages which usually generate an error or refuse to compile if the expected type doesn't match.
- Prototype-based, it is the action of inheriting from other objects but also maintaining its own attributes.
- Multi-paradigm, is simply a way to classify programming languages.
